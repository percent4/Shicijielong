### Shicijielong

使用Python爬虫爬取诗歌作为数据，完成诗词接龙项目，生成exe文件，可直接运行，不依赖任何环境。

生成的exe文件为Poem_Jielong.exe，位于dist/Poem_Jielong.exe,网址为：https://github.com/percent4/Shicijielong/tree/master/dist 。
